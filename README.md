# CodeAlpha Basic Network Sniffer

## Description
A Python-based network sniffer developed for the CodeAlpha Cyber Security Internship.

## Features
- Captures live packets
- Displays source and destination IPs
- Detects TCP/UDP protocols
- Logs packet information
- Real-time monitoring

## Technologies Used
- Python
- Scapy
- Colorama
